特定の見出しを抽出してリストにしてみるテスト

jQuery.indexNav.js

プラグインの概要

説明

特定のIDがついている要素を抽出して、リストに置き換えて表示します。
スムーズスクロールもします。
スクロールの挙動がおかしいのでもう少し練りたい。

オプション指定により見出しテキストを連番に置き換えることや、表示するエリアの指定、スクロールの速度を変えたりすることも可能です。
オプションの詳細については後述します。
入れ子には対応していません。

使用方法

必要なファイル

  • jquery.js
  • jquery.indexNav.js
  • 読み込ませるHTMLファイル

基本の使い方

htmlを構成し、抽出するすべての見出しが入っている要素にidを振ります。
今回は分かりやすいようにidを「post」としました

次にリンクリストを表示するエリアを指定します。
中身は空でいいです。
デフォルトの設定では「id="#linkarea"」になっています。

そして見出しとなる要素にidを振っていきます。
デフォルトの設定ではh2タグです。

<div id="listarea"></div>
<div id="post">
        ・
        ・
        ・
    <h2 id="element01">ひとつ目の見出し</h2>
        ・
        ・
        ・
    <h2 id="element02">ふたつ目の見出し</h2>
        ・
        ・
        ・
    <h2 id="element03">みっつ目の見出し</h2>
        ・
        ・
        ・
</div>

「jquey.js」と「jquey.indexNav.js」を外部スクリプトとしてHTMLのヘッダー内に書きます。

<script src="./jquery.js" type="text/javascript"></script>
<script src="./jquery.indexNav.js" type="text/javascript"></script>

プラグインを動作させるための記述をします。

<script type="text/javascript">
$(function(){
    $('#post').indexNav();
});
</script>

これで見出しが抽出されてリンクされます。
リンクの整形はCSSでご自由にどうぞ。

オプション

オプションの指定の仕方

第一引数にオブジェクトでオプションを渡すことができます。
複数オプションはカンマ区切りで記述してください。

<script type="text/javascript">
$(function(){
    $('#post').indexNav({
        Key: Value
    });
});
</script>

抽出に関するオプション

Key Value Description
indexElement 'h2' 見出しを指定します。
抽出されるのはidがあるものだけです。
escList '.esclist' idがある見出しの中で除外するものを指定します。
除外されるのはこの値がついているものだけです。
listarea '#listarea' リンクリストを生成するエリアのIDを指定します。
listNum false trueにするとリンクリストのテキストが連番に置き換えられます。

スクロールに関するオプション

Key Value Description
scrollSpeed 'fast' スクロールするスピードを指定します。
ミリ秒、または「'fast' / 'slow'」等で指定してください。
ミリ秒の場合はクオーテーションはいりません。
easing 'swing' アニメーションの種類を指定します。
現在は'swing'と'linear'しか指定できません。
Easing Pluginを連携させることも可能です。
scrollTop false trueにすると、ページの先頭に戻るリンクが生成されます。
scrollTop false trueにすると、ページの最下部にスクロールするリンクが生成されます。
scrollTopElement null ページの先頭に戻るボタンに任意のid指定をします。
上部には戻りたいけれど、一番上まで行かなくてもいい、そんなときに。
id指定する場合はクオーテーションで囲んでください。
scrollBottomElement null ページの最下部にスクロールするボタンに任意のid指定をします。
下にスクロールしたいけど、フッター長すぎなんです、的なときに。
id指定する場合はクオーテーションで囲んでください。
scrollTopText 'Go Page Top' ページの先頭に戻るボタンのテキストを任意で指定します。
クオーテーションで囲んでください。
scrollBottomText 'Go Page Bottom' ページの最下部にスクロールするボタンを任意で指定します。
クオーテーションで囲んでください。
scrollPosition 1 ページの先頭に戻るボタンの位置リストの上部に置くか下部に置くかを指定できます。
1を指定した場合(または何も指定しない場合)はリストの上部に、2を指定した場合はリストの下部に生成されます。

裏オプション

Key Value Description
listCreate true falseにすると、そもそもリストを出力しない設定になります。 TOPへ戻るボタンだけ欲しいとかいう場合にどうぞ。

オプションは今後アップデートで追加する予定です。
